# Skyrim Web RPG — Dragonborn Legend

> **Slogan**: "Two Faces, One Memory" | 双面一心，记忆如新

**🌍 Languages**: [English](README.md) | [中文](README.zh.md)

A complete **Skyrim-inspired web RPG** built with pure HTML/CSS/JavaScript. Experience the legend of the Dragonborn right in your browser!

---

## 🎮 Features

### Core Systems
| System | Status | Highlights |
|--------|--------|------------|
| Character Creation | ✅ | 10 races with unique bonuses |
| Main Quest | ✅ | 10 chapters (Helgen → Sovngarde) |
| Side Quests | ✅ | 5 factions (Companions/Thieves/Mages/Dark Brotherhood/Civil War) |
| Combat | ✅ | Turn-based + Shouts + Spells + Items |
| Map System | ✅ | 20 locations with travel encounters |
| Inventory | ✅ | Weapons/Armor/Potions/Shops |
| Save System | ✅ | 3 save slots + auto-save |
| Mobile Support | ✅ | Perfect responsive design |

### Game Content
- **10 Chapters** of epic main storyline
- **5 Faction Questlines** with interwoven plots
- **20+ Discoverable Locations** across Skyrim
- **14 Enemy Types** (Wolf, Bandit, Draugr, Dragon, Alduin...)
- **10+ Weapons & Armor** tiers
- **8 Spells** + **5 Shouts** (Thu'um)

## 📖 Game Guide

### Races
| Race | Bonus | Playstyle |
|------|-------|-----------|
| Nord | HP+20, Frost Res+30% | Warrior |
| Imperial | Speech+10, Gold+50 | Trader |
| Breton | MP+30, Magic Res+15% | Mage |
| Dunmer | Destruction+10, Fire Res+30% | Spellsword |
| Altmer | MP+50, Enchant+10 | Pure Mage |
| Bosmer | Archery+10, Sneak+10 | Archer |
| Khajiit | Sneak+15, Unarmed+15 | Rogue |
| Argonian | Lockpick+10, Poison Res+50% | Thief |
| Orc | Heavy Armor+10, Strength+15 | Tank |
| Redguard | One Hand+10, Stamina+20 | Duelist |

### Combat
- **Attack** - Basic melee/ranged attack
- **Power Attack** - High damage, costs 20 Stamina
- **Block** - Reduce incoming damage
- **Spells** - Destruction (fire/frost/shock) + Restoration (heal/ward)
- **Shouts** - Ultimate abilities (Fus Ro Dah, Yol Toor Shul...)
- **Items** - Potions for HP/MP/Stamina recovery

### Shouts (Thu'um)
| Shout | Words | Effect | Cooldown |
|-------|-------|--------|----------|
| Unrelenting Force | Fus Ro Dah | Push enemies | 20s |
| Fire Breath | Yol Toor Shul | Fire damage | 30s |
| Frost Breath | Fo Krah Diin | Freeze enemies | 25s |
| Slow Time | Tiid Klo Ul | Slow time | 40s |
| Dragonrend | Joor Zah Frul | Force dragons to land | 15s |

## 🎯 Main Quest Chapters

1. **Helgen Escape** - Dragon attack, choose your path
2. **Whiterun** - Report to Jarl, Western Watchtower
3. **Dragon Soul** - Absorb dragon soul, become Dragonborn
4. **High Hrothgar** - Meet the Greybeards
5. **Bleak Falls Barrow** - Retrieve Dragonstone
6. **Diplomatic Immunity** - Infiltrate Thalmor Embassy
7. **Alduin's Wall** - Learn the prophecy
8. **Elder Scroll** - Learn Dragonrend shout
9. **The World's Throat** - Battle Alduin
10. **Sovngarde** - Final confrontation
