# 天霜纪 — 龙裔传说

> **Slogan**: "双面一心，记忆如新" (Two Faces, One Memory)

**🌍 语言**: [English](README.md) | [中文](README.zh.md)

一款完整的**上古卷轴风格网页 RPG**，使用纯 HTML/CSS/JavaScript 打造。在浏览器中体验龙裔的史诗传奇！

---

## 🎮 核心功能

### 游戏系统
| 系统 | 完成度 | 亮点 |
|------|--------|------|
| 角色创建 | ✅ 100% | 10 种族，各有特色加成 |
| 主线剧情 | ✅ 100% | 10 章完整史诗（海尔根→索恩加德） |
| 支线任务 | ✅ 100% | 5 大阵营（同伴团/盗贼公会/法师学院/黑暗兄弟会/内战） |
| 战斗系统 | ✅ 100% | 回合制 + 龙吼 + 法术 + 物品 |
| 地图系统 | ✅ 100% | 20 个地点，旅行遇敌 |
| 物品装备 | ✅ 100% | 武器/护甲/药水/商店系统 |
| 存档系统 | ✅ 100% | 3 存档位 + 自动存档 |
| 移动端适配 | ✅ 100% | 完美响应式设计 |

### 游戏内容
- **10 章主线** 完整史诗剧情
- **5 大阵营** 支线任务线
- **20+ 可探索地点** 遍布天霜省
- **14 种敌人** （野狼、强盗、尸鬼、巨龙、奥杜因...）
- **10+ 武器护甲** 品质等级
- **8 种法术** + **5 种龙吼**
- **随机任务** 无限可玩性

### 战斗系统
- **攻击** - 基础近战/远程攻击
- **强力攻击** - 高伤害，消耗 20 耐力
- **格挡** - 减少 incoming 伤害
- **法术** - 毁灭系（火/冰/电）+ 恢复系（治疗/结界）
- **龙吼** - 终极技能（不屈意志、烈焰吐息...）
- **物品** - 药水恢复生命/魔力/耐力

### 龙吼 (Thu'um)

## 🏗️ 技术栈

- **纯 HTML5** - 无框架依赖
- **CSS3** - 自定义龙裔主题，响应式设计
- **原生 JavaScript** - 零依赖
- **LocalStorage** - 本地存档
- **移动优先** - 触屏友好 UI

## 🎯 主线剧情章节

1. **海尔根之劫** - 巨龙袭击，选择你的道路
2. **白漫城求援** - 向领主报告，西部瞭望塔
3. **龙魂觉醒** - 吸收龙魂，成为龙裔
4. **高赫斯加之路** - 拜见灰胡子
5. **荒芜坠落墓穴** - 取回龙之石碑
6. **外交豁免** - 潜入塞尔摩大使馆
7. **奥杜因之墙** - 解读远古预言
8. **上古卷轴** - 学习龙族末日
9. **世界之喉** - 决战奥杜因
10. **索恩加德** - 最终对决
